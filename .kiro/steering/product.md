# OpenSearch UI Framework (OUI)

OUI is a React component library for building user interfaces in the OpenSearch Project ecosystem. It provides a comprehensive design system with reusable UI components that remove CSS complexity from the development process.

## Key Goals

- Provide reusable React UI components for any OpenSearch web project
- Serve as single source of truth for design aesthetic
- Enable designers to make changes directly in code
- Deliver stable, well-tested UI component API
